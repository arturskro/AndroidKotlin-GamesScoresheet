# AndroidKotlin-GamesScoresheet
Moja w pełni autorska aplikacja. Notatnik dla wyników z gier planszowych itp.
- zachowuje wyniki rozegranych gier
- umożliwia wprowadzenie graczy i przypisywanie ich do gier
- sumuje wyniki graczy z każdej rundy oraz prezentuje je w formie graficznej
Uwagi techniczne
- aplikacja korzysta z bazy danych Room
